# ♟️ Swiss-System Tournament Manager (C++)

A command-line based **Swiss-system tournament manager** written in C++.
This project manages player registration, round pairings, results entry, standings, and tournament persistence.

⚠️ **Project Status:** Active Development
Future versions will include a **web interface** and improved usability.

---

# 🚀 Features

### Player Management

- Add players manually
- Load players from a file
- Withdraw players during the tournament
- Unique player ID system

### Tournament Engine

- Swiss-system style pairing
- Random pairing for Round 1
- Score-based pairing for later rounds
- Prevents repeat matches when possible
- Automatic bye assignment for odd number of players

### Color Balancing

- Tracks white/black history
- Prevents long color streaks
- Attempts fair color distribution

### Results Processing

- Enter results interactively
- Supports:
  - White win
  - Black win
  - Draw

- Automatic score updates

### Tie-Break System

- Median **Buchholz scoring** implemented

### Standings Table

Displays:

- Rank
- Player name
- Score
- Buchholz score
- Status
- Player ID

### Save & Load Tournament

- Resume tournaments anytime
- Stores:
  - scores
  - opponents
  - color history
  - status

# 🧠 Pairing Logic

The system attempts to follow common Swiss tournament rules:

1. Players sorted by
   - Score
   - Buchholz
   - ID

2. Avoid repeat opponents

3. Color assignment rules:
   - Prevent 3 same colors in a row
   - Balance white/black difference

4. Bye rules:
   - Given to lowest eligible player
   - A player receives at most one bye
